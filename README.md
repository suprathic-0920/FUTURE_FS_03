## 📝 Project Pitch: Digital Transformation
**Objective:** To build a premium digital presence for SUPRA Restaurant that not only showcases the brand but directly drives customer acquisition and operational efficiency.

### How this website helps the business grow:
* **Gamified Customer Acquisition:** Implemented an interactive "Scratch Card" system using HTML5 Canvas that securely offers a 50% discount code, highly incentivizing first-time visitors to place an order.
* **Automated Operations:** Built an interactive "Visual Table Reservation" system where users can click and book specific tables (e.g., VIP Lounge, Window Seat). This reduces phone call volumes and automates the booking pipeline.
* **Enhanced Customer Engagement:** Integrated a custom "AI Recommender Bot" that suggests dishes based on user moods, and an immersive UI featuring 3D Tilt Menu cards and a Soulful BGM player. This increases user retention time on the website.
* **Dynamic Personalization:** Engineered a smart time-based JavaScript greeting (e.g., "Flavors of the Night" vs "Rise and Dine") to create a tailored user experience that aligns perfectly with dining psychology.

## 💻 Tech Stack Used
* HTML5
* CSS3 (Custom Dark Theme, Keyframe Animations, Responsive Design)
* Vanilla JavaScript (DOM Manipulation, Canvas API, Event Listeners)
* External Libraries: AOS (Scroll Animations), Vanilla-Tilt.js (3D Cards)
